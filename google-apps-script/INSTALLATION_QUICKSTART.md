# ⚡ WAAS - Szybka instalacja (5 minut)

## Krok 1: Otwórz Google Apps Script
👉 https://script.google.com → **Nowy projekt**

## Krok 2: Wklej skrypt
1. Usuń zawartość `Code.gs`
2. Otwórz plik `setup.gs` z tego folderu
3. Skopiuj **CAŁĄ** zawartość
4. Wklej do edytora
5. **Zapisz** (Ctrl+S)

## Krok 3: Uruchom instalację
1. Wybierz funkcję: `installWAAS`
2. Kliknij: **Uruchom** ▶️
3. **Autoryzuj**: Przejrzyj uprawnienia → Zaawansowane → Przejdź do projektu → Zezwól

## Krok 4: Dodaj klucze API
1. Kliknij: **⚙️ Project Settings** (lewy panel)
2. Przewiń do: **Script Properties**
3. Dodaj właściwości:

```
DIVI_API_USERNAME       → netanaliza
DIVI_API_KEY            → 2abad7fcbcffa7ab2cab87d44d31f5b16b8654e4
PA_API_ACCESS_KEY       → [Twój klucz Amazon]
PA_API_SECRET_KEY       → [Twój sekret Amazon]
PA_API_PARTNER_TAG      → [Twój tag partnera]
```

## Krok 5: Otwórz arkusz
Skopiuj URL z okna dialogowego i otwórz arkusz.

## Krok 6: Gotowe! 🎉
- Menu: **⚡ WAAS**
- Dodaj stronę: **Sites → Add New Site**
- Importuj produkty: **Products → Import from Amazon**
